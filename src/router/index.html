Documentation otakudesu api by onesytex

API Path

https://api-onesydesu.vercel.app/

API Version

v1.0

All Ongoing Anime

Get all ongoing anime

  • [pageNumber] = 1-3 (may change depending on the database)
  • Endpoint: /api/v1/ongoing/[page]
  • Example: https://api-onesydesu.vercel.app/api/v1/ongoing/1

All Complete Anime

Get all complete anime

  • [page] = 1-46 (may change depending on the database)
  • Endpoint: /api/v1/completed/[page]
  • Example: https://api-onesydesu.vercel.app/api/v1/completed/1

Search Anime

Search anime by title

  • [q] = any string
  • Endpoint: /api/v1/search/[q]
  • Example: https://api-onesydesu.vercel.app/api/v1/search/black%20clover

All Anime List

Get all anime list

  • Endpoint: /api/v1/anime-list
  • Example: https://api-onesydesu.vercel.app/api/v1/anime-list

Anime Detail

Get anime detail

  • [endpoint] : String
  • Endpoint: /api/v1/detail/[endpoint]
  • Example: https://api-onesydesu.vercel.app/api/v1/detail/chaisaw-sub-indo

Batch Link

Get batch link

  • [endpoint] : String
  • Endpoint: /api/v1/batch/[endpoint]
  • Example: https://api-onesydesu.vercel.app/api/v1/batch/hakumao-s2-batch-sub-indo/

Genre List

Get genre list

  • Endpoint: /api/v1/genres
  • Example: https://api-onesydesu.vercel.app/api/v1/genres

Genre Page

Get anime by genre

  • [genre] : String
  • [page] : 1 -... (depend on genre)
  • Endpoint: /api/v1/genres/[genre]/[page]
  • Example: https://api-onesydesu.vercel.app/api/v1/genres/action/1
